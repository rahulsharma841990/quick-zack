# Changelog

All notable changes to this project will be documented in this file.

## [1.2.2] - 2026-04-02

### Added
- Copy and paste functionality in terminal

## [1.2.1] - 2026-04-02

### Fixed
- SSH terminal issues - improved shell access from integrated terminal

### Added
- Integrated SSH terminal support via .vscode/sftp.json
- Google Analytics integration
- macOS support with Apple Silicon (arm64) and Intel (x64) builds
- New landing page with modern design

### Improved
- Electron 28 upgrade
- Build system enhancements with GitHub Actions CI/CD
- Download links and documentation updates

## [1.1.0] - 2025-04-01

### Added
- Initial public release
- Windows support (NSIS installer and portable)
- Fuzzy search with Fuse.js
- Multi-editor support
- System tray integration
- Configuration via JSON
